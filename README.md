AdminUserManagement App
//-----------------------------------------------

AdminUserManagement is an app for user management. A user can register an account and login to it. The user can edit their details, reset password, upload profile image etc. 
Admin can manage all the users, including add/delete users, edit user details, block or unblock users. 

Key concepts in this project:
1.Clean architecture (Repository pattern)
2.Authentication (Email/Password + Google OAuth)
3.Authorization (Admin vs User)
4.File uploads (Profile picture)
5.Role-based access
6.Scalable folder structure
7.Type safety end-to-end
